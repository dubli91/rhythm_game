# 플레이 옵션

> 플레이 옵션은 채보의 표시·배치 방식을 플레이어가 조정하게 한다.

## 배경

IIDX 경험의 핵심인 하이스피드·SUDDEN+(레인커버)·RANDOM/MIRROR·오토플레이를 제공한다.
옵션은 곡 선택 화면의 옵션 패널에서 설정하고, 일부는 플레이 중에도 조정 가능하다.

## 요구사항

### 하이스피드 — MUST

1. 노트 낙하 속도 배율 0.50 ~ 10.00, 0.25 단위. 기본값 1.00.
2. 스크롤 속도는 `현재 BPM × 하이스피드`에 비례한다 (IIDX 방식 — 변속 시 체감 속도가 변함).
3. 플레이 중 조정 가능: 기본 키 `PageUp`/`PageDown`(±0.25). 조정 즉시 반영되고 화면에 현재 값을 잠시 표시한다.
4. 설정값은 저장되어 다음 플레이에 유지된다.

### SUDDEN+ (레인커버) — MUST

5. 레인 상단을 커버로 가려 노트 가시 구간을 줄인다. 커버 높이 0~80% (1% 단위).
6. ON/OFF 토글과 커버 높이를 플레이 중 조정 가능: 기본 키 `Home`(토글), `ArrowUp`/`ArrowDown`(높이 ±1%, 누르고 있으면 연속 조정).
7. 커버 높이(화이트 넘버에 해당하는 %값)를 커버에 표시한다.
8. 설정값 저장.

### 배치 옵션 — MUST

9. `OFF / RANDOM / MIRROR` 중 택일. 스크래치 레인은 영향받지 않는다.
   - RANDOM: 건반 1~7 레인을 곡마다 무작위 순열로 치환.
   - MIRROR: 건반 1~7을 좌우 반전.
10. 배치는 판정·게이지에 영향 없는 순수 레인 치환이며, 결과 화면과 기록에 사용 옵션을 표기한다.

### 오토플레이 — MUST

11. 모든 노트를 자동으로 정확히 연주하는 데모 모드. 판정 표시·게이지는 동작하되 **기록은 저장하지 않는다**.
12. 곡 선택 옵션 패널에서 진입한다.

### SHOULD

13. 그린 넘버: 현재 하이스피드·커버 설정에서 노트가 보이는 시간(ms)을 계산해 표시. 변속 곡에서 기준 BPM은 최빈/최대 BPM 중 택일해 명시.
    - 구현 결정 (2026-07-17): 기준 BPM = **최대 BPM**. 선곡 화면 카탈로그에는 곡 단위 BPM min/max만 존재하고(차트 JSON은 플레이 시점에 lazy-load), 최대 BPM 기준은 변속 곡에서 가장 빠른 구간도 읽히도록 하이스피드를 맞추는 보수적 선택이다. 선곡 화면 옵션 패널의 GREEN 표시가 이 값을 쓴다.
    - 플레이 중 HUD의 GREEN 표시는 **현재 BPM**을 따라 실시간 갱신된다 — 스크롤 속도가 `현재 BPM × 하이스피드` 비례(MUST 2)이므로 실제 가시 시간이 변속과 함께 변하기 때문. SUDDEN+ 커버가 켜져 있으면 가시 구간에서 커버 높이만큼 제외하고 계산한다.
14. LIFT(판정선 올리기), S-RANDOM은 차기 버전 후보 — 옵션 패널 구조는 확장 가능하게.

### 그린 넘버 고정 — MUST

15. 그린 넘버 고정(플로팅 하이스피드) 모드: 곡 선택 옵션 패널에서 ON/OFF 토글과 목표
    그린 넘버(ms)를 설정한다. ON이면 하이스피드를 직접 지정하는 대신, 가시 시간이 목표값이
    되도록 유효 하이스피드를 현재 BPM과 SUDDEN+ 커버 높이로부터 자동 유도한다
    (SHOULD 13 가시 시간 정의의 역산). 변속(BPM 변경)·커버 높이 변경 시 즉시 재유도되어
    체감 가시 시간이 일정하게 유지된다. 기본값은 OFF이며, OFF일 때는 MUST 1~4의 수동
    하이스피드가 그대로 동작한다.
16. 목표 그린 넘버는 200~2000ms 범위, 10ms 단위. 고정 모드 중 플레이 내
    `PageUp`/`PageDown`(MUST 3)은 하이스피드 대신 목표값을 ±10ms 조정한다
    (PageUp = 더 빠르게 = 목표값 감소). ON/OFF와 목표값은 저장되어 다음 플레이에
    유지된다(MUST 4와 동일 정책).
17. 유도된 하이스피드가 MUST 1의 범위(0.50~10.00)를 벗어나면 경계값으로 클램프한다.
    이때 실제 가시 시간은 목표값과 달라질 수 있으며, HUD의 GREEN 표시(SHOULD 13)는
    항상 실제 값을 보여준다. 고정 모드는 표시에만 영향을 주고 판정·게이지에는 영향이 없다.

## 수용 기준

- [ ] BPM 150, 하이스피드 2.00일 때와 BPM 300, 하이스피드 1.00일 때 노트 체감 속도가 같다.
- [ ] SUDDEN+ 60%에서 노트가 화면의 아래 40% 구간에서만 보인다.
- [ ] RANDOM으로 플레이해도 총 노트 수·판정 결과 집계가 원본과 동일 조건이다 (레인만 바뀜).
- [ ] 플레이 중 하이스피드를 바꿔도 판정 타이밍은 변하지 않는다 (표시만 변경).
- [ ] 오토플레이 종료 후 해당 곡의 기록이 갱신되지 않는다.
- [ ] 그린 넘버 고정 ON(목표 500ms)으로 변속 곡(예: 140→175)을 플레이하면 변속 순간에도
      HUD GREEN이 500을 유지한다 (유효 하이스피드가 자동으로 바뀜).
- [ ] 고정 모드에서도 동일 입력 기준 판정·게이지 결과는 수동 모드와 동일하다.
