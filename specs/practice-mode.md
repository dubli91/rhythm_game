# 연습 세션

> 연습 세션은 몇 마디 분량의 노트 패턴을 작성해 반복 연습하게 한다.

## 배경

인터뷰에서 요청된 기능: 특정 패턴(트릴, 계단, 스크래치 복합 등)을 몇 마디 작성하고
반복 연습할 수 있는 세션. 곡 전체가 아닌 짧은 패턴의 루프 연습에 특화한다.
판정 엔진·렌더러는 본 게임과 동일한 것을 재사용한다 — 연습이 실전과 같은 감각이어야 한다.

## 요구사항

### 패턴 에디터 — MUST

1. 그리드 에디터: 세로 = 시간(마디/분할), 가로 = 8레인(스크래치+건반 7). 클릭/키보드로 노트 배치·삭제.
2. 마디 수 1~8, 분할(스냅) 4/8/12/16/24/32분음표 선택.
3. BPM 직접 입력 (60~400).
4. 편집 중 패턴은 내부 채보 포맷([chart-format](chart-format.md))의 부분집합으로 표현한다 (BPM 변속·STOP 없음, tap 노트만).

### 연습 재생 — MUST

5. 작성한 패턴을 **끊김 없이 루프 재생**한다. 각 루프 시작 전에 1마디 카운트인(메트로놈 클릭 4박)을 넣는다.
6. 음원이 없으므로 메트로놈 클릭(박자 강/약 구분)을 오디오 클럭 기준으로 정확히 스케줄한다([audio-playback](audio-playback.md)).
7. 판정은 본 게임과 동일한 엔진·윈도우를 사용한다([judgement-scoring](judgement-scoring.md)). 게이지는 없다.
8. 루프마다 통계를 표시한다: 판정별 카운트, 정확도(%), δ 평균(빠름/느림 경향). 루프 간 누적 통계도 표시.
9. 연습 중 BPM을 즉시 변경할 수 있다 (느리게 시작해 점차 올리는 연습 흐름). 하이스피드·SUDDEN+도 본 게임과 동일하게 동작한다.
10. `Escape`로 편집 화면 복귀, 편집↔연습 전환이 3초 이내여야 한다.

### 패턴 저장 — MUST

11. 패턴에 이름을 붙여 로컬(IndexedDB) 저장, 목록에서 불러오기·삭제.

### SHOULD

12. 프리셋 패턴 제공: 트릴(2레인 교대), 계단(1→7), 동시치기, 스크래치+건반 복합 등 기본 연습 세트.
13. δ 히스토그램 표시 (판정 분포 시각화). 확정된 루프들에 걸친 누적 분포로, ±75ms를 10ms 폭 15개 버킷으로 나눈다 (범위 밖 값은 양 끝 버킷에 클램프). 연습 HUD 통계 블록에 한 줄 텍스트 스파크라인으로 렌더링한다.
14. 루프 횟수 지정 후 자동 종료 + 세션 요약.

### 레인 뒤섞기 — MUST

15. 연습 세션에서 건반 1~7의 배치를 직접 지정하는 **뒤섞기** 기능: 1~7이 정확히 한 번씩
    쓰인 7자리 순열 문자열을 입력해 적용한다. i번째 숫자 d는 "화면상 i번 레인에 원본
    d번 레인의 노트를 표시"를 뜻한다 — `1234567`은 원본 그대로, `7654321`은
    MIRROR([play-options](play-options.md) MUST 9)와 동일한 효과. 스크래치 레인은
    영향받지 않는다.
16. 순열이 아닌 입력(자릿수 부족·초과, 숫자 중복, 1~7 밖 문자)은 거부하고 이유를 표시하며
    현재 배치를 유지한다. 적용된 배치는 **다음 루프 시작부터** 반영되고, 현재 배치 문자열을
    연습 HUD에 표시한다.
17. **되돌리기**: 마지막으로 적용한 뒤섞기를 취소해 직전 배치로 복귀한다. 반복 실행하면
    세션 시작 배치(원본 `1234567`)까지 되돌아간다. 되돌리기도 다음 루프 시작부터 반영된다.
18. 배치는 판정·통계에 영향 없는 순수 레인 치환이다([play-options](play-options.md)
    MUST 10과 동일 원칙). 패턴 저장 데이터(MUST 11)에는 포함하지 않으며 세션 동안만
    유지된다 — 편집 화면으로 복귀하면 원본 배치로 초기화된다.

## 수용 기준

- [ ] 4마디 트릴 패턴을 작성 → 저장 → 재로드 → 연습 시작이 키보드/마우스로 1분 내 가능하다.
- [ ] BPM 120에서 루프 경계를 넘어갈 때 메트로놈·노트 타이밍에 끊김이나 드리프트가 없다 (10루프 연속).
- [ ] 연습 중 BPM을 120→180으로 올려도 다음 루프부터 정확히 반영된다.
- [ ] 연습 결과는 곡 기록([results-records](results-records.md))에 저장되지 않는다.
- [ ] 계단(1→7) 패턴에 `7654321`을 입력하면 다음 루프부터 7→1 계단으로 표시되고,
      되돌리기 실행 시 그다음 루프부터 원본 배치로 복귀한다.
- [ ] `1234566`·`123456`·`123456A` 같은 비순열 입력은 거부되고 기존 배치가 유지된다.
- [ ] 뒤섞기 적용 전후로 동일 타이밍 입력의 판정·통계 결과가 배치와 무관하게 동일하다
      (레인만 바뀜).
